## ✨ Features

| Feature | Description |
|---------|-------------|
| **Typewriter Animation** | Hero section cycles through roles dynamically |
| **Dark / Light Mode** | Toggle between themes, preference persists |
| **Particle Background** | Interactive particle network with mouse reactivity |
| **Project Filters** | Filter projects by Python, SQL, ML, Streamlit |

## 🚀 Run Locally

```bash
# Clone the repository
git clone https://github.com/Vanshsrivastava09/portfolio.git
cd portfolio

# Create virtual environment (optional)
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/Mac

# Install dependencies
pip install -r requirements.txt

# Run the app
python app.py
```

Open [http://localhost:5000](http://localhost:5000)
